== Description ==

This Plugin show custom content to the beginning of each post.

For more information, check out <a href="http://www.danycode.com/beginning-post-content/">Beginning Post Content</a>

Plugin Features:

- Customizable content.
- Customizable css style.

== Installation ==

1. Upload the Beginning Post Content plugin to your blog and activate it. 
2. Customize the plugin through: Setting -> Beginning Post Content
